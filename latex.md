# 公式编辑要求

> [!IMPORTANT]
> 遵循以下规则有助于你的公式正确渲染

### 行内数学公式前后空一格

`句子(注意空一格) $math$ (空一格)句子`

`句子(注意空一格) $math$, 这里使用半角的标点所以不用空`

`$math$: 这里使用半角的标点所以不用空`

### 行间数学公式前后空一行

```markdown
段落
(注意空一行)

$$
math
$$

(空一行)
段落
```

## 多行公式

### 方程组

使用 & 对齐

```markdown
f(x)=
\begin{cases}
1,& x \geq 0 \\
114514,& x < 0
\end{cases}
```

$$
f(x)=
\begin{cases}
1,& x \geq 0 \\
114514,& x < 0
\end{cases}
$$

\end{bmatrix}

### 一列整齐且居中的方程式序列

使用 & 对齐

```markdown
\begin{aligned}
f(x) & = (m+n)^2 \\
     & = m^2+2mn+n^2 \\
\end{aligned}
```

$$
